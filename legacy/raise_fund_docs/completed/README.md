## 💼 Company Overview Summary

### **Moonspace - AI-Powered Financial Intelligence Platform**

**Problem**: Australian companies report using AASB standards, creating barriers for global investors who need IFRS/US GAAP formats for comparison and analysis.

**Solution**: AI-powered platform automatically converts Australian financial reports (AASB) to international standards (IFRS/US GAAP) within minutes.

**Market**: $2.8B financial reporting software market with 2,200+ ASX companies and growing international investment demand.

**Business Model**: SaaS subscriptions ($99-2,999/month) + usage-based pricing ($50/report) + professional services.

**Team**:
- **Mike Graham (CEO)**: Serial entrepreneur, 10+ years fintech
- **Tony Pham (CTO)**: 8+ years blockchain/fintech, technical leadership
- **Nguyen Thao Nguyen (CFO)**: MS Quantitative Finance, securities experience

**Traction**: MVP 80% complete, pilot customers identified, technical architecture validated.

**Funding**: Seeking $750K Pre-Seed for 18-month runway to $1.2M ARR.

**Vision**: Become the global standard for financial reporting conversion and analysis.

## 🛠️ Technical Requirements

### Viewing the Pitch Deck
The pitch deck is created using Slidev (https://sli.dev/), a presentation framework for developers.

#### **Option 1: Online Viewing** (Recommended)
1. Upload `pitch-deck.md` to any Slidev-compatible platform
2. Share link with investors for interactive viewing
3. Supports presenter mode and PDF export

#### **Option 2: Local Setup**
```bash
# Install Slidev globally
npm install -g @slidev/cli

# Navigate to the document folder
cd raise_fund_docs/completed

# Start the presentation
slidev pitch-deck.md

# Export to PDF (optional)
slidev export pitch-deck.md
```

#### **Option 3: PDF Conversion**
For investors who prefer PDF format:
1. Use any Markdown to PDF converter
2. Maintain slide structure with page breaks
3. Ensure images and formatting are preserved
